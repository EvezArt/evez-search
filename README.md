# EVEZ Search — Federated Search Engine

Self-hosted federated search powered by SearXNG. Zero tracking, zero cost.

## What It Does
- Federated search across 70+ engines
- Zero tracking, zero logging
- SearXNG instance on :8888 (Docker, restart=always)
- API endpoint for ClawBreak and Cognition integration

## Quick Start

```bash
git clone https://github.com/EvezArt/evez-search.git
cd evez-search
pip install -r requirements.txt
python search.py
```
